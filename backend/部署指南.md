# Windows 台式机部署指南

## 📦 需要拷贝的文件

将以下文件夹拷贝到 Windows 台式机：

```
backend/
├── app.py           # 后端服务
├── install.bat      # 安装脚本
├── start.bat        # 启动脚本
└── README.md        # 使用说明
```

## 🚀 部署步骤

### 1. 在 Windows 台式机上

1. 创建文件夹：`C:\soda567-backend`
2. 将 `backend` 文件夹里的所有文件复制进去
3. 双击运行 `install.bat`（首次安装，需要 10-20 分钟）
4. 安装完成后，双击 `start.bat` 启动服务

### 2. 测试服务

1. 打开浏览器访问：http://localhost:5000/api/health
2. 如果看到 `{"status":"ok"}` 说明服务正常

### 3. 更新网站

将 `tools/transcribe.html` 上传到 GitHub：

```bash
cd /Users/mac/Documents/Claude_WorkSpace/soda567
git add tools/transcribe.html tools/index.html
git commit -m "添加视频/音频转文案工具"
git push
```

## 🌐 使用流程

1. **在台式机上启动后端服务**（运行 start.bat）
2. **访问网站**：https://soda567.dpdns.org/tools/transcribe.html
3. **上传文件**，等待处理
4. **复制文案**

## ⚠️ 注意事项

- 后端服务必须在台式机上运行
- 台式机需要联网
- 首次使用会自动下载模型（约 150 MB）
- 处理速度：1 分钟音频 ≈ 5-10 秒（RTX 2060）

## 🔧 高级配置

### 允许局域网访问

如果想在其他设备上使用，需要：

1. 修改 `transcribe.html` 第 88 行：
```javascript
const API_URL = 'http://你的台式机IP:5000/api/transcribe';
```

2. 在 Windows 防火墙添加端口 5000 的入站规则
